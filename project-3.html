Project Overview

This Project is a Lightweight Analytics Reporting Service API built with Nodejs, Express.js and MongoDB that implements API Endpoints for Storing and Fetching the IP and Coordinates. It also calculates the distance of the path that is traced through the coordinates stored for the IP over the last one hour.
